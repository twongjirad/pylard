# pylard 2: The Lardening

Again! But this time, better!

## python Liquid Argon (Event) Display 2

Goal is to provide a visual algorithm development framework using data products from LArCV and LArLite

## Dependencies

* ROOT
* Cython
* PyQtGraph
* LArLite
* LArCV

## Installation

* install ROOT
* install LArLite
* install LArCV
* install sip
* install PyQt4
* install PyOpenGL
* install virtualenv
* make environment
* run setup.py
